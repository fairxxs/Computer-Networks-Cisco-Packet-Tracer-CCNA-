## 📁 Lab Reports

### Lab 1 - Basic Router Configuration & LAN Connectivity
**File:** `Lab_Report_Router_Config_and_LAN.pdf`

- Configured Cisco 1941 router via CLI (hostname, passwords, MOTD banner, NVRAM)
- Set up GigabitEthernet and Serial interfaces for LAN connectivity
- Verified end-to-end connectivity across multi-router topology
- Used: `show running-config`, `show ip interface brief`, `copy run start`

---

### Lab 2 - IPv4 Subnetting & VLSM Design
**File:** `Lab_Report_Subnetting_VLSM.pdf`

- Designed subnetting scheme for 192.168.0.0/24 network
- Calculated optimal subnet mask (/26) supporting 50+ hosts across 4 subnets
- Implemented Variable Length Subnet Masking (VLSM) for efficient IP allocation
- Built and verified topology in Cisco Packet Tracer

---

### Lab 3 — IPv6 Addressing & Device Configuration
**File:** `Lab_Report_IPv6_BasicConfig.pdf`

- Configured global unicast and link-local IPv6 addresses on router interfaces
- Enabled IPv6 routing (`ipv6 unicast-routing`) across multiple network segments
- Connected 6 end devices (Sales, Billing, Accounting, Design, Engineering, CAD)
- Verified end-to-end IPv6 connectivity with `show ipv6 interface brief`

---

### Lab 4 - Wireshark Ethernet Frame Analysis
**File:** `Lab_4716_Wireshark_Ethernet_Frames_Report.pdf`

- Captured and analyzed Ethernet II frames using Wireshark
- Examined ARP request/reply frames: MAC addresses, frame types, FCS
- Analyzed local traffic (ping to gateway) vs remote traffic (ping to cisco.com)
- Identified frame fields: Preamble, Destination/Source MAC, Frame Type, Data, FCS
